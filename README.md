## Data
All videos in this benchmark can be downloaded via [this link](https://drive.google.com/file/d/1z1bF6csd4ZRRbhSV7DAoCsUqj2CbFQi_/view?usp=drive_link). After downloading, please either put the videos to the [./data/Videos](./data/Videos) directory or change the "--dataset_dir" argument in the code. These videos are sourced from the following human action datasets:
- [NTU RGB+D 120](https://rose1.ntu.edu.sg/dataset/actionRecognition/)
- [UCF-101](https://www.crcv.ucf.edu/data/UCF101.php)
- [ActivityNet](http://activity-net.org/)
- [Kinetics-700](https://github.com/cvdfoundation/kinetics-dataset)
  
The benchmark utilizes two separate annotation pipelines for its two constituent parts. All annotations are stored in the [./data/GT](./data/GT) directory.

## Running
Please run this code via <code>main.py</code>. The arguments can be specified either through the [YAML](./configs/Qwen2.5-VL-7B-Instruct.yaml) file (in [./configs/](./configs/)) or the command line. 
```bash
python main.py -c Qwen2-VL-7B-Instruct -e
python main.py -m Qwen2-VL-7B-Instruct -s NTU -g 0,1 -e
```
Arguments specified via the command line will take precedence over those defined in the YAML file.
